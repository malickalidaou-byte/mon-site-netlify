# Mon Site Netlify

Mon premier site web hébergé sur Netlify.

## Comment déployer sur Netlify

1. Connecte ton compte GitHub à Netlify
2. Sélectionne ce repo
3. Netlify détecte automatiquement les paramètres
4. Clique sur "Deploy"

## Fichiers

- `index.html` - Page principale
- `style.css` - Styles
- `script.js` - Interactions JavaScript
